# Claude Skills

A collection of agent skills for [Claude Code](https://claude.ai/code) and other skills-compatible agents.

Install with:

```bash
npx skills add g4berolo/claude-skills
```

## Skills

### buddy-reset

Don't vibe with your current buddy? Reroll and hatch a new one!

This skill resets your Claude Code companion, giving you a fresh chance to meet a new little friend. Just say "reroll buddy" and restart Claude Code — then run `/buddy` to see who shows up.

**Triggers:** `buddy reset`, `reroll buddy`, `new buddy`, `change buddy`, `reset companion`
